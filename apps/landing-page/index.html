What you get

Everything Git needs.
Nothing it doesn't.

Built for developers who want power without complexity.

Visual Git Graph

Interactive commit history with branches, tags, diffs, and a clear visual layout. Browse your entire repo history at a glance.

AI Commit Messages

Powered by a local Ollama LLM. Your diff never leaves your machine. Conventional commits, streamed live.

Safe Rollback

Revert, soft/mixed/hard reset with visual previews and confirmation gates. Undo history across all destructive actions.

Working Tree

Stage, unstage, commit, push and pull. WIP batch-commit mode with per-group AI messages.

GitHub Integration

Device-flow OAuth, pull request views, SSH key management. No passwords stored.

English & French

Full i18n support in English and French. More languages welcome via contribution.

Your data is yours

Zero telemetry.
Full privacy.

Runs entirely on your machine

No cloud backend. No account required. Just your local git repos.

No analytics, no trackers

We don't know who you are, how you use the app, or what repos you open.

Credentials stay in Rust

SSH keys and HTTPS tokens never reach the JavaScript layer. Tauri ACL enforced.

AI runs on Ollama — locally

Your code diffs go to localhost:11434, not to OpenAI or any cloud.
